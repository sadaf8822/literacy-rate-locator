# Literacy Rate Data Visualizer
Demo folder consists of demo videos. Video-1: Code and Video-2: Output and explanation of no need to install any dependencies.
An interactive web application designed to visualize and filter literacy rate data across different regions using an integrated map interface. 

This project is built as a **standalone frontend application**, meaning it runs directly in your web browser with no complex setup or backend server required.

## Key Features
* **Interactive Map:** Powered by Leaflet.js for smooth geographical visualization.
* **Smart Filtering:** Dynamic sliders to filter data by Literacy Rate and Age.
* **Quick Search:** Search and locate specific individuals/records by name.
* **Portable:** Zero dependencies to install; just open and run.

## How to Execute
Because this application is built to be portable, you do not need to install Node.js or download any external libraries.

1. **Clone or Download** this repository to your local machine.
2. **Ensure** that `index.html` and `Literacy_rate.csv` are in the same folder.
3. **Open** the `index.html` file in any modern web browser (Google Chrome, Microsoft Edge, or Firefox). or in vscode, run go live
4. **Upload** your `Literacy_rate.csv` file using the "Upload" button provided in the dashboard.
5. **Interact** with the map, filters, and search bar to explore the data.

## Technologies Used
* **React:** For managing the application state and components.
* **Leaflet.js:** For rendering the interactive map and markers.
* **Babel Standalone:** To enable React JSX syntax directly in the browser.
* **HTML5/CSS3:** For the dashboard layout and styling.
